# Lakepointe / Josh Howerton — Curated Search Domains

**Purpose:** A hand-picked domain list for a Google Programmable Search Engine (PSE / CSE) restricted to "search only included sites," to surface mentions of **Lakepointe Church** and **Josh Howerton** — including critical/negative coverage and the Christian-nationalism angle driven by the *Live Free* podcast.

**Research date:** 2026-07-08.
**Count:** 63 domains (target was ≥50, so there's room to trim). Categorized below.

---

## How to use this in a Google Programmable Search Engine

1. In the PSE control panel, set the engine to **"Search only included sites."**
2. Add each domain below. To capture *all* pages on a site, add it as a URL pattern: `christianpost.com/*` (a bare `christianpost.com` entry also works and includes subdomains).
3. **Exclude your own properties** so self-mentions don't flood results: do **not** add `lakepointe.church` or `joshhowerton.com`. (Add them only if you specifically want to monitor your own output.)
4. Run your queries with the person/entity as an exact phrase, e.g. `"Josh Howerton"`, `"Lakepointe Church"`, `"Live Free" Howerton`. Pair this with the `NEGATIVE_CONTEXT_TERMS` filter in `lakepointe-disambiguation.md` for the ambiguous "Lake Pointe" name collisions.

> Reality check on coverage: outlets in the **Confirmed** section were each verified (2026-07-08) to have published about Howerton/Lakepointe. Outlets in the other sections were selected because they own this exact beat — SBC/evangelical news, church-accountability watchblogs, Christian-nationalism reporting, and DFW local news — but I did **not** individually confirm a Howerton article on each. A few (e.g. Rolling Stone, The Guardian) may not have covered him *yet*; they're included because they're the natural home for bigger national/critical coverage if a story breaks.

## Notes & caveats

- **Confirmed vs. beat:** Only Section 1 (14 domains) is verified coverage. Sections 2–5 are curated by beat relevance, not individually confirmed — expect some to have zero Lakepointe hits until/unless a story lands there.
- **Google PSE domain cap:** Programmable Search Engine comfortably supports far more than 50 included sites, so the full 63 will work; trim Section 6 first if you want to keep it lean.
- **Christian-nationalism angle:** The *Live Free* podcast episodes ("4 Megachurch Pastors Discuss Christian Nationalism," "Should We Make America a 'Christian Nation' Again?") are the main hook for critical secular coverage — that's why Section 4 leans national/political.
- **Not included on purpose:** your own domains (`lakepointe.church`, `joshhowerton.com`) and pure aggregator/AI-wiki pages (e.g. Grokipedia), which aren't original reporting.
- **Pair with the disambiguation filter:** run these searches alongside the `NEGATIVE_CONTEXT_TERMS` list in `lakepointe-disambiguation.md` so the generic "Lake Pointe" collisions (GA sports complex, AL state park, the Michigan Lakepointe Church, apartments, etc.) don't create false positives.
